# Chakshu Saini — Portfolio

A simple, static personal intro site built from my resume. No backend required.

## Deploy to GitHub Pages

1. Create a new repository (e.g. `username.github.io` or `resume-page`).
2. Push this folder to the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, set source to **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder, then save.
6. Your site will be live at `https://username.github.io/repository-name/` within a few minutes.

## Local preview

Open `index.html` in your browser, or run a simple server:

```bash
npx serve .
```

## Files

- `index.html` — main page
- `styles.css` — styling
- `script.js` — mobile nav and footer year
- `Chakshu_New.pdf` — downloadable resume
